Maikados

Über…

Maikados ist eine Brettspielumsetzung einer empfehlenswerten Vorlage. Es handelt sich um eine Multiplayervariante, die zum Testen von verschiedener Technologie entwickelt wurde. Daher wird ein Server benötigt. Den gesamten Source-Code gibt es auf GitHub.

Es handelt sich um eine Implementierung zu privaten Zwecken, da keine Rechte vorhanden sind.
